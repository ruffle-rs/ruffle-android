Voodoo Steuern / Gamepad & Tilt Mapper

Since v.20, Magic DOSBox supports external controllers. It's possible to use various controllers like an xbox 360 or Nvidia Wireless controller. Their buttons can be mapped to DOSBox keys, mouse buttons, native gamepad buttons or special actions.
Since v47 Magic Dosbox supports two players. To get it work, enable "Two players" option and setup players. Please press "Assign players" from General Settings for assigning gamepads to player profile. Two players support is disabled by default. Some games detect gamepads at startup. In this case, save settings and restart application.
Joystick
In this section is possible setup various joystick controls like :
•Buttons
•Dpad
•Left joystick
•Right joystick
•L2 and R2 axis
Buttons
Classic buttons what works like a keyboard keys. They have an Up and Down state and can be bound to these actions:
•Keys
•Left and Right mouse click
•Native gamepad buttons
•Show keyboard
•Widgets - currently only deadly strike (macro) is supported
Dpad
Dpad can be controlled with
•KeyPad - standard keyboard keys
•2Axis - support for native 2axis joystick. Dpad does not support fully functional 2axis range, but only edges -1,0,1. This is very useful for game calibration or for old DOS games what don't need full axis range.
Left and Right joystick
Left and right joysticks support these controls :
•KeyPad - standard keyboard keys
•Mouse - mouse support
•2Axis - support for full 2axis range from -1.0 to 1.0
L2 and R2
L2 and R2 are axis. Works similarly to buttons, but currently support only keys.
Tilt
A silly function for those inclined to use it in racing games. ;)
•KeyPad - standard keyboard keys
•2Axis - support for full 2axis range from -1.0 to 1.0
2axis support
Since version 38 was added supports for native 2axis joystick.

These requirements must be meet:

•In game profile must be enabled "Native 2axis joystick" emulation. If the expert settings are enabled then type this in joystick section :

[joystick]
joysticktype=2axis

•In addition many games requires setup joystick in their setup.exe or other configuration options
•Many games will start joystick calibration at startup or in configuration options. It's required for proper joystick ingame functionality
Temporary block
External keyboards connected to Android device usually don't support all keys. Lack of keys like F1, F2 etc might be a critical issue for dos programs. Fortunately Magic Dosbox offers solution. It is possible bind keys 1-0 to F1-F10 and temporarily block/unblock mapper with reserved key.
•Reserved key - temporarily activates/deactivates mapper
While testing was found another issue. External keyboard messed with key presses. Key press means sequence : KEY DOWN and hold down key to next KEY UP. But external keyboard used during development didn't hold key down, instead was sent sequence KEY DOWN, UP, DOWN, UP...for a time while was key pressed. Because of this must be reserved key toggler. Maybe other keyboards will work better and reserved key can disable/enable mapper for a time when is reserved key down. But there must be at least 1 keyboard what really sends standard key sequences.
