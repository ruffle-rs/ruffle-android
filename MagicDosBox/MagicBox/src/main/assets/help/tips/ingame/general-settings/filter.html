HQ2X shader
Please note that HQ2X may not work correctly on all GPUs, and may slow down DOSBox. There is big difference between mobile and desktop graphic cards. While desktop graphic cards can handle the various opengl shaders with ease, mobile GPUs are optimized to be battery friendly and powerful at the same time. The downside of this optimization prevents all shaders to work correctly on mobile devices.
