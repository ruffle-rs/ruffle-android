Advanced Settings

Special settings which are not suitable to be placed anywhere else ;)
Block Mouse Move
Use this flag if the absolute mouse gets desynchronised too quickly, but you want still use it. This flag blocks mouse dragging, but clicking is allowed. This increases the time until the absolute mouse can become desynchronised.
Manual Mouse Correction
This options can fine tune the absolute mouse to make it more accurate.
Correction type
Currently three absolute mouse fixes are supported. By default is set "default" type, because it works for major part of games. Second type is attempt to fix absolute mouse for windows games. Works partially and for games what don't need mouse dragging (use "block mouse move" flag). All types can be affected with "Manual mouse correction". Please see our website for more information.
Speed Patches
Patches for refresh and cycles. They can help when is DOSBox is under stress.
Orientation
Lock screen orientation to landscape or portrait. Default value is rotate.
Run at start
This option allows to fire picked deadly strike or key widget on dosbox start. In other words you can send key sequence or prepare user interface at startup. Useful for games what requires initial input from keyboard.
Show general settings on back button
Default action. Escape key is triggered if this options is disabled. Then you need to use a "special" widget or long-press on the "back" button to bring up General Settings.
IP Address
Show current local ip address. Useful detection if you started ipx server. Only works when connected to network.
