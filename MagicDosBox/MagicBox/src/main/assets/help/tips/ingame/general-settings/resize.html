Resize

Using this option will activate "Aspect Mode" which enables you to modify the screen size. It's possible to change the overall screen size percentage and screen aspect ratio.
These modes are supported
•Fit Screen
•Full Screen
•Custom
Fit screen
Keeps the original game resolution ratio.
Fullscreen
Stretches the game to the size of your screen.
Custom
Special mode. Allows to set custom size for width and height separately up to 150% of "Fit screen" dimensions. Default value for resize is width. Tapping on the title "width 100%" switches to height. This mode allows to move left-top corner of the scene outside visible boundaries.
Scene can be unlocked with long press and moved to new position.
Note : Game profile contains additional flag "aspect ON/OFF". This is native dosbox function and is equivalent to aspect=false/true command.
