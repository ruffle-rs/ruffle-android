Right Mouse Click Support

One of the most frequently asked questions is about how to emulate the right mouse button on a touchscreen. Magic DOSBox supports several options.
Right Click via "Mouse Touch" Widget
You can place this mouse widget anywhere on the screen. This widget is very flexible and allows you to change the mouse button permanently (all subsequent taps will be executed as right clicks), or only when you touch this widget. Using this widget is fast and without delay.
Right Click via Gestures
Gestures can be configured in the "General Settings" menu using the "Gestures and Mouse" options. Gestures require detection time and thus using this mouse mouse method is more or less delayed - depending on the gesture type.
These gestures support right mouse click
•Double Tap
•Two-point Gesture
•Long-press
