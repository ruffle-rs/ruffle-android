Differences between the free and donated version

The free version of Magic DOSBox offers approximately 90% of the functionality of the donated version. Below are the main differences.
•In the free version you're allowed only one game in the collection.
•Regarding the above point, the free version can export but can't import or duplicate game profiles.
•Magic DOSBox supports 11 types of on-screen buttons(widgets). In the free version you can only use 3 types.
Widget list (purple are allowed in the free version)
•Key Widget - keyboard key mapping
•Mouse Touch - mouse touch management
•Mouse Navigation - absolute/relative mouse switch
•Master of Puppets - virtual dpad
•Bag - container for widgets
•Journal - text notes. Great for old-school rpgs!
•Walkthrough - allows to load offline html help files
•Deadly strike - action combo
•Target - automatically click on a pre-set specific place on the screen
•Telescope - screen zooming
•Special - can call specific system functions (for example taking a screenshot)
