Magic Dosbox features

Main menu
•add games to collection (one game profile in free version)
•create game groups
•move games between groups
•export/import/delete game profiles
•create game shortcut on desktop
•specific startup settings for every game
•possibility to set custom text and image
•help system
•global and main menu settings
•multi-language support (slovak, english, german, russian, finnish, croatian, italian, chinese, french)
•last started games panel
•default and custom sizes for file browser, main menu and various in game menus
•sdcard support
In game settings
•unlimited number of on-screen widgets/buttons with custom image, text, background image and custom position on screen. Text and Image inside widget can be resized and positioned by your needs. Support for custom ttf fonts
•various modes, main are design mode and play mode
•customize screen size and position in aspect mode
•dozens painted images and background images for widgets styling. Possibility to add your own
•absolute and relative mouse
•support for samsung stylus include its button
•support for x360 joystick, nvidia shield controller and other external gamepads
•support for 2 players
•support for physical mouse
•support for sound blaster, pc speaker, midi and gus
•mappable swipes gestures
•longpress, double tap, two-point gestures
•support for *.iso, *.gog, *.inst and *cue ogg support
•in-game screenshots with gallery. Useful if you need something remember in adventure or rpg
•fast emulation with plenty optimizations
•orientation lock to landscape or portrait
•support for ipx and playing over LAN
•various graphics filters
•forum and website
•support for android 2.3+
Others
•custom build for arm, arm7, arm7-neon and x86 devices
