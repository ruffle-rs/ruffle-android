Version 1.0.42

Released : 26.10.2016
•Virtual joystick supports full two axis range

•Virtual joystick supports resizing of the inner circle

•Redesigned gestures options

•fixed reported crash – “bad token exception”. Occasionally when user pressed back button on phone, mDosbox crashed. It should be fixed now.

•Key widget can be toggleable

•Fixed resize problem on ChromeBooks (big thanks to Magnus)

•Added support for mounting multiple floppies. “Swap image” can cycle over them. No user interface for now, only console

•Magic Dosbox is now fully controllable with Gamepad! Not tested in window, but probably can work correctly only on fullscreen. And unfortunatelly due various unexpected behaviors in last days of testing, is this feature enabled for Android 5+ devices. Hope older android versions will be supported as well in next version.

•Updated banner for Android TV

•Fixed S-pen issue on note 5 (Thanks Ryan)

•Occasionally Magic Dosbox crashed by polling input events from android to dosbox. This cross-thread crash should be now fixed. Many thanks M-HT for help!

•Flag "Use faster audio" is now controllable by user.

Thanks to M-HT, Alex, Josip Prekratic, Elv, Magnus, Ryan and Thorsten for help with this version. Also thanks to all for support.
