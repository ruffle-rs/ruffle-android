Copying a game to your device

There are multiple ways you can get a game on your device. I won't go over them here. Google is your friend. Here are some hints: you can connect your device to a PC via cable, you can copy a game to your device using wi-fi, or you can just download a game from a web server. I'm going to focus on where the game data should be placed on your device.
Where to place game data?
MS-DOS and Windows set one hard-disk (HDD) as primary. For many users, this is the famous 'C:\' Drive. On this drive you can find folders like "Windows", "Program Files" etc. This drive is very important and we need to tell DOSBox where it's located on Android.
Where's the C:\ Drive?
The C:\ Drive is located whenever you want it -- you can specify the location within DOSBox. For example, it can be the root directory of your device
For example : /storage/emulated/0/
NOTE : this folder varies per device!
We will copy the game data to this folder
Recommended path
Despite the example shown above, it is strongly recommended to not copy games to the root directory of your device. Instead, create a custom folder.
For example : Drive_C
Path will be for example : /storage/emulated/0/Drive_C
