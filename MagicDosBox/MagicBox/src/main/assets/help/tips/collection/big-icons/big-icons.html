Icons too big?

Magic DOSBox sets the default size for menu icons and file manager items to medium. This size works fine for most devices. If you think that this size is too small or too big, you can change it in "Global Settings".
These sizes can be changed :
1. Collection items
Collection game icons. These can be changed under the section "Game collection".
2. File browser items
File manager files and folders. These can be changed under the section "Filebrowser".
3. Various menus and image viewer
The size of menus like "General Settings" can be modified, as well as when you skin a widget with predefined images. These can be changed under the section "Menu and Image Viewer".
