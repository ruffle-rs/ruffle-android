Main menu settings

Main menu specific settings.
Title
Simple description what will be shown in top left corner of main menu.
Items size
Size of icons in main menu. The "Default" value tries calculate best size for your device. This value may be overriden with three other predefined values.
Back to main menu when quits :
Game or MS-DOS application can be started in three ways. First way is simple tap on game from main menu. Second way is from game configuration window using setup button (requires set "Setup" path under "Software" section). And third way is when "Expert settings" are enabled. By default only second way returns you back to main menu. But it's possible change it here.
Show last started games
Shows or hides panel with shortcuts to last started games. Panel is located on right side of main menu.
