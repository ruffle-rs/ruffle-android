Version 1.0.52

Released : 08.05.2017
•fixed crash on devices with small resolution (below 400pixels)

•New feature - Loopers. Loopers can cycle over set of defined keys. Loopers can be configured from general settings and can be controlled with gamepad or with new widget type - "Looper widget". This widget can be bound to swipes, this means, it is possible cycle over keys with swipe gesture.

•added new widget type - Looper

•Removed L2 and R2 axis option from Voodoo control/Gamepad & tilt option. This option was merged with buttons option. It will be much more user friendly. But please setup these L2/R2 again.

•fixed situation when saving global settings reverted material design theme to fantasy

•Item deletion on fonts and voodoo control (gamepad & tilt) options is more user friendly

•In Power/Cycles menu is now possible enter cycles value manually

•Updated translations
