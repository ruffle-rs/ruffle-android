Importing and Exporting Game Profiles

Importing and exporting game profiles was introduced in version 16 and serves for sharing layouts between players. The main goal is to provide game profiles with various fixes which other players can further modify.
Exporting
Use long-press on the added game in the collection and pick "export".
Importing
Press '+" button on left toolbar. Pick "import" option with crate icon from shown dialog and import a game profile.
Import function, including duplication are only available in the donated version.
