Game files

Let's say we've installed our hypothetical game "Battle Tanks" in the folder BTanks. Looking inside the folder we will find these files :
BATTLE.PAK
BATTLE.CFG
BATTLE.EXE
ENGLISH.PAK
FINALE.PAK
INTRO.PAK
OPTIONS.CFG
SCENARIO.PAK
SETUP.EXE
SOUND.PAK
To help you understand the anatomy of a DOS program, I've highlighted the different file types with separate colors.
Brown files that store various data required by game.
Red files that store settings.
Green files that are executable and the most important files.
Executable files
Executable files can end with '.exe', '.com', or '.bat'. Battle Tanks starts with BATTLE.EXE. The second exe file SETUP.EXE starts another program in which you can set various settings such as sound card configuration.
