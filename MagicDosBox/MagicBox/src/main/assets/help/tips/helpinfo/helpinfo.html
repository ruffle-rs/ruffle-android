Help

Help is divided into three sections
•Collection Help
•In-game Help
•Design-mode Help
The appropriate Help section is shown when using the corresponding part of Magic DOSBox and each section can be enabled/disabled separately.
A disabled help section can still be accessed manually. For example, in the collection page, tap on the button in the bottom-right corner. When in-game or in Design-mode, the help section can be accessed via the "General settings" button.
