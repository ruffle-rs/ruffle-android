Key Widget

One of the two most important widgets. Allows you to create a custom keyboard screen overlay. Pressing the widget means "key down" and releasing it means "key up". The Key widget has special gestures.
Gestures
•Single Tap
•Double Tap
If only one key is enabled then the response is quick. The Double tap is useful for saving space on your screen, but firing is delayed based on timing.
Deadly strike and Key widget can be bound to startup. This means that actions will be fired soon after start automatically.
