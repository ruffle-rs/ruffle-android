Editing Widgets

The widget configuration windows has two important parts. The header settings and the widget's specific settings.
Header Settings
Header settings are split into two sections: the first section, "Look and Feel" in which you can change the foreground image, background image, color, opacity and text. The second section, "General" contains flags for controlling widgets.
Flags
•Undetectable - the widget is not recognized by your finger and is invisible. This is useful for mapping.

•Visible - with visibility disabled, the widget is detectable, but is not shown. You can create an invisible Escape key for example.

•Activate on Tap Only - If enabled, the widget can't be activated during a finger drag. Disabling this setting is useful for arcade games.

•Deactivate on Leave - the widget is deactivated when your finger leaves its borders when your finger drags on the screen. Disabling this can be useful for the "Master of Puppets" widget.
