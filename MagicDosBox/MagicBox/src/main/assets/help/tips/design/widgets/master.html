Master of Puppets

A virtual dpad able to send keystrokes, mouse events and can control native dpad.
Type
•4 way - only up, down, left and right directions are recognized
•8 way
•mouse - moves absolute mouse over screen. Game must fully support absolute mouse
•2axis - supports native 2axis gamepad.
2axis
This type allows bind virtual joystick to native 2axis gamepad. This type must meet these requirements :

•In game profile must be enabled "Native 2axis joystick" emulation. If the expert settings are enabled then type this in joystick section :

[joystick]
joysticktype=2axis

•In addition many games requires setup joystick in their setup.exe or other configuration options
•Many games will start joystick calibration at startup or in configuration options. It's required for proper joystick ingame functionality
Flags
•Diagonals ON - cross directions will fire both keys together. For example an upper-left movement will fire both : the left + up arrows together.
•Activate Dpad When You Are Outside Circle - the dpad can be activated by hitting outside the circle, but inside the dpad.
